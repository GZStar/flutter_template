# flutter_template
Flutter project template

本项目为个人学习Flutter的练习项目。

基本包含了日常开发中常用的组件。

同时使用了Getx，使整个项目框架目录结构清晰，非常适合学习借鉴。

# 项目运行环境
```Flutter
1. Flutter version 3.13.6
2. Dart version 3.1.3
```
